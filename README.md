# UbuntuCare.africa - Healthcare Registration Platform

A modern, responsive healthcare registration platform built with HTML, CSS, and JavaScript.

## Features

- **Patient Registration Form**: Clean, user-friendly registration interface
- **Responsive Design**: Works seamlessly on desktop and mobile devices
- **Modern UI**: Built with Tailwind CSS for a professional appearance
- **Healthcare Focus**: Designed specifically for UbuntuCare.africa's patient registration needs

## Technologies Used

- HTML5
- Tailwind CSS
- Font Awesome Icons
- Google Fonts (Inter & Playfair Display)

## Live Demo

View the live registration form at: [https://didi-skipper.github.io/Ubuntucare/](https://didi-skipper.github.io/Ubuntucare/)

## Getting Started

1. Clone the repository
2. Open `ubuntucare.html` in your web browser
3. Fill out the registration form
